# Ghi chú thay đổi pair 7

## File

- Bản v2 đã chỉnh sửa: `HỢP ĐỒNG PHÂN PHỐI sản phẩm dinh dưỡngmỹ phẩm (MS 01) v2_modified.docx`

## Ghi chú về định dạng

File gốc là `.doc`; môi trường hiện không có công cụ chuyển đổi `.doc` sang `.docx`, và pipeline hiện chỉ hỗ trợ `.docx`/`.pdf`. Vì vậy bản v2 được tạo dưới dạng DOCX mẫu rút gọn dựa trên chủ đề hợp đồng để phục vụ dữ liệu thực nghiệm. Các dòng dưới đây mô tả rõ nội dung được thiết kế trong v2 để tạo các trường hợp thay đổi.
